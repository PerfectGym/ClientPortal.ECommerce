Embeding demo with return url

If you want user to be redirected back to your page from payment gate then you need to init flow with 3 parameters:

  • flowId - id of the e-commerce flow, this is always a string, you can find it in URL
  • element - html element or CSS selector to render the ecommerce
  • returnUrl - URL that you want to redirect to after payment

If returnUrl wasn't defined then user will end up reservation process on our Thank you page.

<head>
    <!-- PerfectGym.ECommerce styles imports-->
</head>
<body>
    <div id="main"></div>

    <!-- PerfectGym.ECommerce script imports -->

    <script>
        PerfectGym.ECommerce.init({
            flowId: 'b64d45ff3',
            element: '#main',
            returnUrl: 'https://github.com/PerfectGym/ClientPortal.ECommerce'
        });
    </script>
</body>
